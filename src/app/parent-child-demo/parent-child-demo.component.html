ViewChild vs ContentChild - Minimal Example

🔍 Key Differences:

  • ViewChild: References components declared in the parent's template
  • ContentChild: References components projected via <ng-content>

📋 Usage Example:


<app-parent>
  <app-child></app-child>  <!-- This becomes ContentChild -->
</app-parent>

💡 Tip: Open browser console to see the lifecycle logs!
